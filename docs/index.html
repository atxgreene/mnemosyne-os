Quick Start

git clone https://github.com/atxgreene/mnemosyne-os.git
cd mnemosyne-os
./scripts/install-local.sh
./scripts/run-dev.sh

python bin/mnemosyne store "Memory graphs make continuity visible" --domain memory-graph
python bin/mnemosyne route "Synthesize cognitive OS design" --json

Security note: keep the API local-only. Authentication is future work.
